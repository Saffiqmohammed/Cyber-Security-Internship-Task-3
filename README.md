## Objective
The main objective of this task is to learn basic networking concepts by capturing and analyzing live network traffic. This includes understanding how DNS, TCP, HTTP, and HTTPS protocols function and how network communication is secured using encryption.

---

## Key Observations
- **DNS Analysis:**  
  Observed DNS packets that show how domain names are resolved into IP addresses before a connection is established.

- **TCP Traffic:**  
  Analyzed TCP communication and identified the three-way handshake process (SYN, SYN-ACK, ACK), which ensures reliable connections.

- **HTTP vs HTTPS:**  
  HTTP traffic was visible in plain text, while HTTPS and TLS traffic appeared encrypted, highlighting the importance of secure communication.

- **Protocol Filtering:**  
  Used protocol filters such as `dns`, `tcp`, and `http` to efficiently analyze specific types of network traffic.

---

## Security Relevance
Network traffic analysis plays a vital role in cyber security. It helps detect unusual or suspicious activities such as unauthorized access attempts, malware communication, and data leakage. Encryption protects sensitive data during transmission, and DNS monitoring can help identify access to potentially malicious websites.
